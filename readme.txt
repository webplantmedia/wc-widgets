== Installation ==

1. Uzip the `wc-widgets.zip` folder.
2. Upload the `wc-widgets` folder to your `/wp-content/plugins` directory.
3. In your WordPress dashboard, head over to the *Plugins* section.
4. Activate *WordPress Canvas Gallery*.
